Detailed Description

Builder class for BitDict.

Unless you already have the bits stored contiguously, this should be the preferred way to build a BitDict.

Construct a BitDictBuilder with the given number of bits.

Note that the size cannot be changed after construction. The BitDict created by build() will have the same number of bits. All bits are initialized to 0.
